## 菜单模块

### 菜单类型


1. 文件夹
> 左侧菜单栏的目录
2. 页面容器
> 将一个业务的多个功能归类到一个Tabs容器下
3. 页面
> 普通页面
4. 流程
> 指定流程的实例列表页面：如请假流程的实例列表，可视化配置生成列表页面
5. 实体
> 指定实体的实例列表页面：如项目实体的实例列表，可视化配置生成列表页面
6. 按钮
> 页面的子元素，用于前端控制按钮权限
